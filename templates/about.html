Проект Jumoreski API - это приложение, созданное для поиска нужных для Вас юморесок.
Параметры поиска различны - минимальная длина, есть ли картинка, есть ли тредшот, сколько юморесок должно выдаваться и другие.
Хотите найти 10 самых популярных юморесок? Или может быть 25 новейших тредшотов? А как насчёт 5 случайных юморесок с картинкой?
Этот проект поддерживает как графического оформление так и API.
Проект сделали Тимофеев Данил и Алексеев Вячеслав. Выполнено для Яндекс.Лицея
